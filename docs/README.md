# substill — documentation

The rendered documentation site (user guide + autogenerated API reference) is built from
this directory with Sphinx and published to GitHub Pages. Build it locally:

```bash
pip install -e ".[docs]"
python -m sphinx -b html docs docs/_build/html   # open docs/_build/html/index.html
```
